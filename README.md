# Adidas-US-Interactive-Sales-Dashboard

## Project Overview
This project is an interactive sales dashboard built using Microsoft Excel to analyze Adidas US sales data.  
The dashboard provides detailed insights into sales performance, profit analysis, regional trends, and product-level performance.

The objective of this project is to transform raw sales data into meaningful business insights using data visualization techniques.

## Tools & Technologies Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning & Transformation
- Calculated Fields

## Dataset Information
The dataset includes the following fields:
- Retailer
- Region
- State
- Product
- Sales Method
- Units Sold
- Total Sales
- Operating Profit
- Operating Margin


## Dashboard Features

### Sales Analysis
- Total Sales by Region
- Sales by Retailer
- Sales by Product Category

### Profit Analysis
- Operating Profit by Region
- Profit by Sales Method
- Margin comparison across products

### Performance Metrics
- Total Sales
- Total Profit
- Units Sold
- Average Operating Margin

### Interactive Filters
- Region Slicer
- Retailer Slicer
- Product Slicer
- Sales Method Filter


## Data Cleaning Steps
- Removed blank rows
- Standardized text formatting
- Checked for duplicate records
- Verified numeric columns
- Created calculated profit metrics


## Key Insights
- Certain regions generate significantly higher revenue.
- Some retailers outperform others in profitability.
- Online sales show different profit margins compared to in-store sales.
- Product-level analysis helps identify high-performing categories.


## Business Impact
This dashboard helps:
- Identify top-performing regions and retailers
- Improve sales strategy
- Optimize profit margins
- Support data-driven decision making


## Conclusion
This project demonstrates how Excel can be used to create interactive business dashboards and generate actionable insights from sales data. It highlights practical data analysis skills useful for real-world business scenarios.
